   - 2.3 One-Way Function

     * These are functions that are easy to compute in one direction but potentialy hard to reverse. Technically reversing the function would ideally take millions of years even using all the world's computers.
     * Furthermore, there is no mathematical proof that one-way functions exist, but many functions (like x^2 in a finite field) behave as if they are.
     * pecial type where reversing the function is easy only if you possess a specific secret piece of information the trapdoor.


    -  2.4 One-Way Hash Functions

        * Takes a variable length input and converts it into a fixed length output string called a hash.
        * One Wayness: It is easy to calculate the hash from the input but computationally unfeasible to find the input given a hash value.
        * Changing just one bit of the input changes approximately half of the bits in the resulting hash value.
        * It's purpose is to provide a way to verify the integrity of a file or message without needing the entire file.
